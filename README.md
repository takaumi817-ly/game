# にほんご N3 ことばげーむ

ちゅうごくご ぼご むけ の にほんご がくしゅうげーむ です。  
JLPT N3 レベルの ごいを、**めいし / どうし / けいようし / ふくし** で ばらんすよく しゅつだいします。

## とくちょう

- にほんごの ひょうきは **かなのみ**（かんじなし）
- 4たく くいず（ことばの いみ を えらぶ）
- ききとり くいず（おとを きいて ことばを えらぶ）
- すこあ / れんぞくせいかい を ひょうじ

## つかいかた

1. このりぽじとりで いかを じっこう:
   ```bash
   python3 -m http.server 8000
   ```
2. ぶらうざで `http://localhost:8000` を ひらく
3. くいずに こたえて がくしゅうかいし

# 日本語 N3 ことばゲーム

中国語母語話者向けの日本語学習ゲームです。
JLPT N3 レベルの語彙を、**名詞 / 動詞 / 形容詞 / 副詞** でバランスよく出題します。

## 特徴

- 出題文はかな表記（学習者向け）
- UI の説明文やステータス表示は漢字を含む表記
- 4択クイズ（単語の意味を中国語で選ぶ）
- 聞き取りクイズ（音声を聞いて中国語の選択肢から選ぶ）
- 聞き取り問題では日本語選択肢を表示しない
- スコア / 連続正解を表示

## 使い方

1. このリポジトリで以下を実行:
   ```bash
   python3 -m http.server 8000
   ```
2. ブラウザで `http://localhost:8000` を開く
3. クイズに答えて学習開始

# JLPT N3 日语词汇训练（面向中文母语者）

这是一个给中国学习者使用的日语单词记忆小游戏：
- 题目语言为中文说明 + 日文词汇。
- 词汇难度定位在 **JLPT N3**。
- 题库按词性均衡：**动词 / 形容词 / 副词 / 名词**。

## 使用方式

1. 在仓库根目录运行：
   ```bash
   python3 -m http.server 8000
   ```
2. 打开浏览器访问：`http://localhost:8000`
3. 开始练习 20 道 N3 词汇题。

## 规则

- 每题 4 选 1，选择最准确的中文意思。
- 答对 +10 分。
- 实时显示连对次数、当前词性与总进度。
- 可随时点击“重新开始”重新洗牌练习。

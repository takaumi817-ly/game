# 日本語 N3 ことばゲーム

中国語母語話者向けの日本語学習ゲームです。
JLPT N3 レベルの語彙を、**名詞 / 動詞 / 形容詞 / 副詞** でバランスよく出題します（語彙を増量）。

## 特徴

- 出題文はかな表記（学習者向け）
- UI の説明文やステータス表示は漢字を含む表記
- 4択クイズ（単語の意味を中国語で選ぶ）
- 聞き取りクイズ（音声を聞いて中国語の選択肢から選ぶ）
- 聞き取り問題を含め、選択肢は中国語のみ
- 1プレイ最大100問
- 選択肢は毎問で重複しないように生成
- スコア / 連続正解を表示

## ファイル構成

- `index.html`: 画面のHTML構造のみ
- `style.css`: デザイン・レイアウト・配色などのスタイル
- `words.js`: 単語データ（かな・中国語訳・品詞）
- `app.js`: クイズ生成、選択肢生成、スコア計算、ボタン操作などのゲームロジック

## 使い方

1. このリポジトリで以下を実行:
   ```bash
   python3 -m http.server 8000
   ```
2. ブラウザで `http://localhost:8000` を開く
3. クイズに答えて学習開始


## words.js のデータ構造

`words.js` の各語彙データは次のオブジェクト形式です。

- `jp`: 日本語（漢字かな交じり）
- `kana`: 読み（かな）
- `zh`: 中国語訳（簡体字）
- `pos`: 品詞（`名詞` / `動詞` / `形容詞` / `副詞`）
- `topic`: 学習トピック（例: `学校` / `仕事` / `生活` / `自動詞・他動詞` / `中国語と意味がずれる語`）
- `trap`: 中国語母語話者が混同しやすいポイント

例:

```js
{
  jp: "予定",
  kana: "よてい",
  zh: "计划；预定",
  pos: "名詞",
  topic: "時間",
  trap: "予約・約束と混同しやすい"
}
```

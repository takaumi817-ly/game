# 日本語 N3 語彙ゲーム

中国語母語話者向けの日本語学習ゲームです。  
JLPT N3 レベルの語彙を、**名詞 / 動詞 / 形容詞 / 副詞** でバランスよく出題します。

## 特徴

- 出題する日本語語彙は **かな表記のみ**
- UI は読みやすさのため通常の日本語（漢字あり）
- 4択クイズ（日本語語彙の意味を中国語で選択）
- 聞き取りクイズ（音声を聞いて中国語の単語を選択。選択肢に日本語は表示しない）
- スコア / 連続正解 / 聞き取り問題数を表示

## 使い方

1. このリポジトリで以下を実行:
   ```bash
   python3 -m http.server 8000
   ```
2. ブラウザで `http://localhost:8000` を開く
3. クイズに回答して学習開始

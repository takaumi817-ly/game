# JLPT N3語彙ゲーム 設計メモ

## 目的
- 本メモは、JLPT N3語彙ゲームに今後追加する語彙設計を整理するための設計資料。
- 今回は設計のみを行い、ゲーム本体（`index.html` / `app.js` / `words.js` / `style.css`）は変更しない。

## 1. JLPT N3語彙ゲームに入れるべき単語カテゴリ

### 1-1. 品詞カテゴリ（基本分類）
- 名詞
- 動詞
- 形容詞
- 副詞（必要に応じて表現語も含める）

> まずは上記4分類をベースにし、出題モード（意味選択・文脈選択）で学習効果を高める。

### 1-2. 場面カテゴリ（N3実用語彙）
- 学校
- 仕事
- 生活
- 買い物
- 交通
- 人間関係
- 感情
- 時間
- 原因・結果

> 品詞分類に加えて場面タグを付与し、同一語の多面的な出題を可能にする。

## 2. 品詞カテゴリ：名詞、動詞、形容詞、副詞

### 名詞
- 例: 予定、経験、原因、結果、習慣、関係、場合
- 用途: 意味選択問題に向く。読解語彙の基礎固め。

### 動詞
- 例: 増える、減る、比べる、選ぶ、続ける、断る
- 用途: 活用・文脈依存を含む問題に向く。

### 形容詞
- 例: 詳しい、必要、十分、危険、自然、複雑
- 用途: 評価・状態・程度の理解に有効。

### 副詞
- 例: だんだん、ぜひ、ほとんど、ついに、かなり
- 用途: 聴解・読解での意味判断に重要。

## 3. 場面カテゴリ：学校、仕事、生活、買い物、交通、人間関係、感情、時間、原因・結果

### 学校
- 例: 授業、試験、成績、提出、説明、確認、復習

### 仕事
- 例: 連絡、会議、予定、残業、担当、報告、相談

### 生活
- 例: 食器、洗濯、掃除、修理、故障、準備、片付け

### 買い物
- 例: 値段、割引、支払い、注文、返品、交換、無料

### 交通
- 例: 乗り換え、遅れる、到着、出発、混む、案内

### 人間関係
- 例: 相手、関係、約束、招待、断る、謝る、頼む

### 感情

### 時間
- 例: 最近、将来、途中、たまに、しばらく、いつか

### 原因・結果
- 例: 原因、理由、結果、影響、ため、せい、場合

## 4. 中国語母語話者向けの注意点
- 漢字から意味推測できる強みがある一方、「分かったつもり」による誤答が起こりやすい。
- したがって、以下を重点設計とする。
  - 中国語と意味がずれる漢字語
  - 似ているが使い分けが違う語
  - 自動詞・他動詞ペア
  - かなだけで提示したときに難しくなる語
- 出題では「日本語→中国語」だけでなく「中国語→日本語」「文脈穴埋め」も想定する。

## 5. 中国語と意味がずれる漢字語

| 日本語 | 中国語での誤解例 | 日本語での実際の意味（中国語） |
|---|---|---|
| 手紙 | トイレットペーパーと誤解 | 信、手纸 |
| 勉強 | 無理に強いる意味と混同 | 学习 |
| 丈夫 | 夫関連語と誤解 | 结实、健康 |
| 新聞 | ニュース媒体全般と誤解 | 报纸 |
| 汽車 | 自動車と誤解 | 火车、列车 |
| 愛人 | 配偶者・恋人と誤解 | 情人、不倫相手 |

## 6. 似ているが使い分けが違う語
- 約束 / 予約 / 予定
- 参加 / 出席 / 通う
- 知る / 分かる / 覚える / 思い出す
- 直す / 治す / 修理する
- 上がる / 上げる
- 下がる / 下げる

> 類義語群は単語単体ではなく、短文での使い分け出題を前提にする。

## 7. 自動詞・他動詞

| 自動詞 | 他動詞 |
|---|---|
| 開く | 開ける |
| 閉まる | 閉める |
| 入る | 入れる |
| 出る | 出す |
| 始まる | 始める |
| 決まる | 決める |
| 変わる | 変える |
| 集まる | 集める |

- 将来の出題案（例）: 「ドアが ___」→ 開く / 開ける の文脈選択。

## 8. かなだけだと難しい語

| かな | 漢字 | 中国語 |
|---|---|---|
| けいけん | 経験 | 经验 |
| よてい | 予定 | 计划、预定 |
| げんいん | 原因 | 原因 |
| けっか | 結果 | 结果 |
| せつめい | 説明 | 说明 |
| かくにん | 確認 | 确认 |
| しゅうかん | 習慣 | 习惯 |

- かな表示時は、同音異義・意味近接語との混同対策として文脈情報を付与する。

## 9. 今後 words.js に入れる単語データ構造案

> 実装変更は今回行わない。以下は将来拡張用の設計案。

### 推奨フィールド案
- `id`: 一意ID
- `jp`: 日本語見出し（通常表記）
- `reading`: かな
- `zh`: 中国語訳（主要義）
- `pos`: 品詞（noun / verb / adjective / adverb）
- `scene`: 場面タグ配列（school, work, life, shopping, transport, relationship, emotion, time, cause_effect など）
- `difficulty`: JLPT難易度（N3）
- `notes`: 注意点（誤解しやすい意味差、使用条件）
- `confusable`: 混同しやすい語ID配列
- `transitivityPair`: 自他対応語ID（存在する場合）
- `kanaRisk`: かな提示時の難易度フラグ
- `example`: 例文
- `exampleZh`: 例文中国語訳

### JSONイメージ（将来案）
```json
{
  "id": "n3_001",
  "jp": "予定",
  "reading": "よてい",
  "zh": "计划；预定",
  "pos": "noun",
  "scene": ["work", "time"],
  "difficulty": "N3",
  "notes": "予約との使い分けに注意",
  "confusable": ["n3_057", "n3_103"],
  "transitivityPair": null,
  "kanaRisk": true,
  "example": "来週の予定を確認します。",
  "exampleZh": "我确认下周的计划。"
}
```
